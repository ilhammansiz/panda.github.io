ABOUT

GitHub is a code hosting platform for version control and collaboration. lets you and others work together on projects from anywhere. This tutorial teaches you GitHub essentials like repositories, branches, commits, and Pull Requests. For most beginners, Github may simply be a platform to upload code, but its simply much more than that. It can easily be used as a collaboration platform among coders, and can be used to build complex systems. For a beginner, its recommended to start learning the programming syntax first.
